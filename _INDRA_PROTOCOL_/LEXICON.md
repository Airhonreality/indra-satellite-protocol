# 📖 INDRA LEXICON : EL LENGUAJE DE LA MATERIA

Este diccionario formaliza los verbos y conceptos del sistema. El uso de terminología genérica (get/set/save) es un vector de entropía. Usa estos términos para mantener la alineación.

| Término Genérico | Verbo Indra | Significado en el Ecosistema |
| :--- | :--- | :--- |
| `save()` | `Crystallize()` | El acto de convertir una intención del satélite en realidad física en el Core. |
| `getData()` | `Resonate()` | El acto de pedir información al Core y esperar a que la UI vibre con la respuesta. |
| `init()` | `Ignition()` | El proceso de arranque donde el satélite se identifica y handshackea con el Core. |
| `sync()` | `Resonance()` | El estado de equilibrio entre la verdad del Core y la proyección del Satélite. |
| `delete()` | `Ghosting()` | Marcar algo como inexistente pero manteniendo su rastro para no romper la integridad. |
| `module` | `Slice` (Rebanada) | Una funcionalidad vertical completa (desde el esquema hasta el botón). |
| `database` | `Silo` | Un lugar físico de almacenamiento (Drive, Notion, etc.) gestionado por un Provider. |

## 🧘 CONCEPTOS DE ALTO NIVEL
*   **Materia Oscura**: Datos hardcodeados, variables globales sin esquema o lógica que no pasa por el Bridge.
*   **Espina Dorsal**: La carpeta `src/score/schemas/`. El origen de toda definición.
*   **Sinceridad**: Cuando la UI refleja exactamente lo que el Core ha validado.
