# Blueprint: Cotizador Web (Front-End General)

**Intención Narrativa**: Embudo progresivo (Lead-magnet paramétrico) de cualificación sutil.

## Lógica y Datos (Engine UI)
- **Agnosticismo Evaluativo**: Decisiones mapeadas a Etiquetas Dinámicas (Tags) en lugar de JSONs duros de base de datos (`Material_Piedra`, `Espacio_GranFormato`).
- **Cálculo Delta Reactivo**: El backend (Indra) exporta Rangos de Valor fijos al cliente. Escoger acabados suma deltas locales (`+$1M COP`) de forma aislada e instantánea mitigando interrupciones por lag de internet.
- **Arquitectura UI**: Formularios ocultos. Flujo sustituido por selecciones táctiles hiper-grandes. Finaliza en reporte de Lead automatizado sincronizando de subida con Base CRM.
